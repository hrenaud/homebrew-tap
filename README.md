# hrenaud/homebrew-tap

Tap Homebrew pour [ai-footprint](https://github.com/hrenaud/ai-footprint) — compteur
d'impact environnemental multi-critères pour outils d'IA agentique.

## Installation

```bash
brew tap hrenaud/tap
brew install ai-footprint
```

Ou en une commande :

```bash
brew install hrenaud/tap/ai-footprint
```

## Documentation

`brew help`, `man brew` ou la [documentation Homebrew](https://docs.brew.sh).
